# Cashbook
一个自用记账本APP

> 运行项目请在 /buildSrc/src/main/kotlin/ 下添加文件 SigningConfigs.kt
> ```kotlin
> object SigningConfigs {
> 
>     /** 密钥别名 */
>     const val keyAlias = ""
> 
>     /** 别名密码 */
>     const val keyPassword = ""
> 
>     /** 密钥文件路径 */
>     const val storeFile = "xxx.jks"
> 
>     /** 密钥密码 */
>     const val storePassword = ""
> }
> ```
